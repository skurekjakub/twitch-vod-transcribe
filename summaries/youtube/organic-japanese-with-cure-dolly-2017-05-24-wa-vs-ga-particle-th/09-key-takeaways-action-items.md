## 9.1. Essential Concepts to Internalize

### The Core Insight

**Wa (は) and ga (が) are NOT two versions of the same thing—they are entirely different TYPES of particles.**

| Particle | Type | Function |
|----------|------|----------|
| が (ga) | Logical particle (格助詞) | Marks the subject; tells us WHO does the action |
| は (wa) | Non-logical particle | Marks the topic; tells us what the sentence is ABOUT |

### The Five Key Principles

1. **Every Japanese sentence has a ga-marked subject** (even when invisible)

2. **Ga binds sentences logically**—changing ga changes WHO does WHAT

3. **Wa creates topic-comment structure**—changing wa only changes focus, not meaning

4. **The zero pronoun (ø)** is the invisible subject that carries the implicit ga

5. **Topic-comment is grammar, but not logic**—は provides grammatical structure without logical function

| Term | Meaning | Notes |

## 9.3. Grammar Structures to Practice

### Structure 1: Basic Subject-Predicate

```
[Subject] が [Predicate]
私が さくらです
"I am Sakura"
```

### Structure 2: Topic-Comment

```
[Topic] は [Comment]
私は さくらです
"As for me, [I] am Sakura"
```

### Structure 3: Full Sentence with Topic

```
[Topic] は + ø が + [Predicate]
土曜日は + ø が + 東京に行きます
"As for Saturday, [I] am going to Tokyo"
```

### Structure 4: Transitive Action

```
[Subject] が [Object] を [Verb]
私が メアリーを キスしました
"I kissed Mary"
```

## 9.4. Suggested Next Steps for Learners

### Immediate Actions

1. **Practice sentence analysis**
   - Take simple Japanese sentences
   - Identify: topic (は), subject (が, visible or ø), predicate
   - Write out the full structural analysis

2. **Create comparison sentences**
   - Write sentences with the same action but different topics
   - Example: 
     - 月曜日は学校に行きます (Monday topic)
     - 学校は月曜日に行きます (School topic)

3. **Review previous knowledge**
   - Revisit any sentences where you used は or が
   - Ask: "Did I use this correctly given the logical vs. non-logical distinction?"

### Short-Term Actions

4. **Watch the next lesson**
   - The instructor promises to cover the specific implications of choosing wa vs. ga
   - This builds directly on the foundational concepts from this lesson

5. **Get the companion book**
   - "Unlocking Japanese" (Amazon, paperback or Kindle)
   - Provides additional practice and reinforcement

### Long-Term Practice

6. **Active reading/listening**
   - When consuming Japanese content, consciously identify:
     - What is the topic?
     - What is the logical subject?
     - How does the sentence structure create meaning?

7. **Production practice**
   - When speaking/writing, consider:
     - Do I need to state a topic, or is it clear from context?
     - Am I marking the correct element as subject vs. topic?

## 9.6. Common Errors to Self-Correct

| Error | Correction |
|-------|------------|
| Thinking は and が do the same job | They are different types of particles with different functions |
| Always starting with 私は | Use zero pronoun when context is clear |
| Believing は changes sentence meaning | は only changes topic/focus, not logical content |
| Forgetting the invisible が | Every sentence has a が-marked subject (ø が if not visible) |
