## 8.1. Instructor Self-Introduction

**Context/Setting:** Beginning of the lesson

**Speaker:** Cure Dolly (instructor)

**Transcript:**

```
Instructor: KawaJapa no Kyua Dorii desu.
Instructor: I am Cure Dolly from KawaJapa.
```

**Analysis:**
- **Japanese:** KawaJapa no Kyua Dorii desu
- **Romanization:** KawaJapa no Kyua Dorii desu  
- **Literal structure:** [KawaJapa の Cure Dolly] です
- **Full analysis:** ø が KawaJapa の Kyua Dorii です

**Key Language Points:**
- Demonstrates natural self-introduction WITHOUT "watashi wa"
- Shows that the zero pronoun (ø) is sufficient when context is clear
- The instructor explicitly notes she does NOT say "watashi wa" because it would add unnecessary emphasis

## 8.2. Example Sentences (Not Dialogues)

This lesson primarily uses isolated example sentences rather than dialogues to illustrate grammatical points.

### Example Set 1: Topic Variation

**Context:** Demonstrating how changing the topic (は) doesn't change the logical action

**Sentences:**

```
Sentence 1: 土曜日は東京に行きます
             Doyoubi wa Toukyou ni ikimasu
             "As for Saturday, I'm going to Tokyo"

Sentence 2: 東京は土曜日に行きます
             Toukyou wa doyoubi ni ikimasu
             "As for Tokyo, I'm going there on Saturday"
```

**Key Language Points:**
- Same logical action (going to Tokyo on Saturday) expressed with different topics
- Changing は from 土曜日 to 東京 changes focus but not meaning
- Both sentences contain an invisible ø が (zero pronoun with ga)

---

### Example Set 2: Identity Statement Variations

**Context:** Demonstrating three ways to express "I am Sakura"

**Sentences:**

```
Version 1 (Zero pronoun): さくらです
                          Sakura desu
                          "[I] am Sakura"
                          Analysis: ø が さくらです

Version 2 (With topic):   私はさくらです
                          Watashi wa Sakura desu
                          "As for me, [I] am Sakura"
                          Analysis: 私は + ø が さくらです

Version 3 (With subject): 私がさくらです
                          Watashi ga Sakura desu
                          "*I* am Sakura" (emphatic)
                          Analysis: 私が さくらです
```

**Key Language Points:**
- Three grammatically correct ways to express the same identity
- Each carries slightly different nuance/emphasis
- The instructor promises to explore these implications in the next lesson

### Example Set 3: Transitive Action

**Context:** Demonstrating the object marker を (wo)

**Sentence:**

```
私がメアリーをキスしました
Watashi ga Mearii wo kisu shimashita
"I kissed Mary"
```

**Structural Analysis:**
- 私が (watashi ga): "I" marked as the doer of the action
- メアリーを (Mearii wo): "Mary" marked as the receiver of the action
- キスしました (kisu shimashita): "kissed" (past tense, polite)

**Key Language Points:**
- Clear demonstration of logical particles working together
- が marks WHO did the action
- を marks WHO received the action
- This is a purely logical sentence (no topic marker は)

## 8.3. Lesson Closing

**Context/Setting:** End of the lesson

**Speaker:** Cure Dolly (instructor)

**Transcript:**

```
Instructor: Thank you for attending this lesson. 
            If you have any questions or comments please put them 
            in the comments below and I will answer them. 
            And remember that the course book for this course of lessons 
            is Unlocking Japanese. You can get it at Amazon in paperback 
            or in Kindle editions, and it goes with this course 
            and will help you to understand everything in it. 
            Thank you again.

Instructor: これからよろしくお願いします。
            Kore kara yoroshiku onegai shimasu.
            "I look forward to our continued relationship."

Instructor: Class dismissed.
```

**Key Language Points:**
- **これからよろしくお願いします (Kore kara yoroshiku onegai shimasu):** 
  - Formal/polite closing expression
  - Literally: "From now on, please treat me favorably"
  - Used to establish ongoing relationship with students
  - Appropriate register for teacher-student context
